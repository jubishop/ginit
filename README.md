# Ginit

Installs opinionated scaffolding for a new Gem.

## Installation

```zsh
gem specificinstall jubishop/ginit
```

## Usage

- Create a new empty git repo, such as `my-repo`.

```zsh
git clone https://github.com/jubishop/my-repo.git
cd my-repo
ginit
```
